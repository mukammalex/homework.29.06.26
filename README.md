

# CreateX News Page

Ин лоиҳа саҳифаи **News**-и вебсайти CreateX мебошад, ки бо истифода аз **HTML5** ва **Tailwind CSS** сохта шудааст.

## Имкониятҳои лоиҳа

- Дизайни Mobile First
- Responsive барои телефон, ва компютер
- Header бо меню ва маълумоти тамос
- Hero Section
- Categories барои хабарҳо
- Blog Cards
- Pagination
- Contact Form
- Footer бо маълумоти ширкат ва обуна ба хабарнома

## Технологияҳои истифодашуда

- HTML5
- Tailwind CSS
- Git
- GitHub

## Қисмҳои саҳифа

### Header
- Логотип
- Менюи навигатсионӣ
- Рақами телефон ва почтаи электронӣ

### Hero Section
- Breadcrumb
- Сарлавҳаи саҳифа
- Расми асосӣ

### Categories
- All News
- Company News
- Innovation
- Industry News
- Expert Tips
- Marketing

### Blog Section
Дар ин қисмат 6 корти хабарӣ ҷойгир шудааст, ки аз инҳо иборатанд:
- Расм
- Сарлавҳа
- Категория
- Сана
- Тавсифи кӯтоҳ

### Pagination
Барои гузаштан байни саҳифаҳои гуногуни хабарҳо истифода мешавад.

### Contact Form
Формаи ирсоли дархост, ки дорои чунин майдонҳо мебошад:
- Ном
- Рақами телефон
- Почтаи электронӣ
- Паём
- Checkbox барои розигӣ

### Footer
- Маълумоти ширкат
- Суроға ва маълумоти тамос
- Истинодҳои муфид
- Обуна ба хабарнома

## Тарҳи Responsive

Лоиҳа бо усули **Mobile First** сохта шудааст ва барои дастгоҳҳои зерин мутобиқ мебошад:

- Телефонҳои мобилӣ
- Компютерҳо
